关于作者

👋 关于我

你好！我是长安，一名对技术充满热忱、以结果为导向的全栈开发者与AI工程师。我坚信技术的价值在于创造可量化的商业成果，并致力于通过代码构建高效、智能的解决方案。

我的学习与实践经历赋予了我从后端到前端的全面开发能力，以及在人工智能领域的深度工程化实践经验。我享受迎接挑战的过程，具备极强的快速学习与技术攻坚能力，并能在高压环境下稳定地推动项目前进。

🚀 我的核心专长

我的技术栈围绕三大核心支柱构建：

全栈应用与系统构建

我具备独立构建复杂Web应用与系统的能力。无论是使用 Flask 或 Django 开发符合MVC/MVT模式的后端服务，还是运用 Vue 生态打造功能丰富、体验流畅的前端界面，我都游刃有余。我的实践不仅限于企业级应用，更延伸至构建高性能搜索引擎，实现了从分布式数据采集、毫秒级检索到智能化前端交互的全流程。

AI与数据工程化

我在人工智能领域的专长不止于理论，更侧重于工程落地。我擅长基于大语言模型 (LLM) 及 LangChain 等框架开发智能应用。同时，我拥有数据工程经验，能够设计并实现高性能、分布式的爬虫架构 (Scrapy)，并利用 Elasticsearch 和 Redis 等技术构建稳定、高效的数据处理与索引链路。

DevOps与安全实践

我熟练驾驭 Linux (Ubuntu) 服务器环境的管理与维护。不仅独立搭建并维护了个人的技术博客 (Hexo + GitHub Pages)，还出于对网络安全的浓厚兴趣，成功部署了Cowrie蜜罐系统进行攻击流量分析。这些实践让我积累了宝贵的系统管理、自动化部署与安全防护实战经验。
